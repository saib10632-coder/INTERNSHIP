# Project Abstract

The **Customer Prepaid Travel Wallet System** is a full-stack web application built for **Manivtha Tours & Travels** to digitise prepaid travel wallet management. Staff use the system to create wallets, record deposits and deductions, monitor auto-deduction limits, and view customer transaction histories. Management accesses dashboards, analytics charts, and exportable reports for business decisions.

**Technology:** React (frontend), Node.js Express (backend), JSON file persistence (local prototype; PostgreSQL for production), Recharts for analytics.

**Users:** Admin staff, operations team, and management at Manivtha Tours & Travels.

**Impact:** After deployment, wallet operations move from manual spreadsheets to a single source of truth — reducing errors, enabling real-time visibility, automating low-balance alerts, and improving customer retention through prepaid wallet lock-in.
